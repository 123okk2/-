# Naver_News_title_classification
네이버 뉴스 제목 크롤링 후 제목에 따라 뉴스 주제 분류 (정치, 경제, 사회, 생활, IT)


# Files
newsTitleCrawling.ipynb : 네이버 뉴스 제목 크롤링 (5개 분야)

modelTraining.ipynb : Konlpy, TFIDF 벡터를 통한 전처리 및 로지스틱 회귀 학습
